Common HTML Tags

<a> Tag

Defines a hyperlink to another page or resource.

<a href="https://www.example.com">Visit Example</a>
Visit Example

<header> Tag

Represents the header of a document or section, typically containing navigation links or introductory content.

<header><h1>Welcome to My Website</h1></header>

Welcome to My Website

<nav> Tag

Defines a set of navigation links.

<nav><a href="#home">Home</a></nav>
Home | About | Contact

<main> Tag

Specifies the main content of a document, unique to that document.

<main>Main content here</main>

Main Content Area

This is the main content of the page.

<article> Tag

Represents independent, self-contained content.

<article><h2>Article Title</h2></article>

Article Title

This is an article content.

<table> Tag

Defines a table for displaying tabular data.

<table><tr><th>Header</th></tr></table>
Name Age
John 25
Jane 28

<ul> Tag (Unordered List)

Defines an unordered list.

<ul><li>Item 1</li></ul>
  • Item 1
  • Item 2
  • Item 3

<ol> Tag (Ordered List)

Defines an ordered list.

<ol><li>First</li></ol>
  1. First
  2. Second
  3. Third

<img> Tag

Defines an image.

<img src="image.jpg" alt="Description">
